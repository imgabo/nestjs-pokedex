# Ejecutar en desarrollo
1. Clonar el repo
2. Ejecutar
```
npm run install
```
3. Tener Nest Cli Instalado
```
npm i -g @nestjs/cli
```

4. Levantar la base de datos
```
docker-compose up -d
```


## Stack Usado
* MongoDB
* Nest
